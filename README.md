# Student-Study-Pal
Student Study Pal is a console-based Python application designed to help students manage and improve their academic performance by organizing study schedules, tracking deadlines, creating quizzes, and monitoring their weekly progress in one centralized system.

**Features:**
	
📅 Study Schedule	

	•	Add study schedules with a date, time, and task description. 
	•	View all saved study schedules. 
	•	Input validation for date and time formats. 

✅ Deadline Tracker

	•	Add academic tasks and deadlines. 
	•	View all saved tasks. 
	•	Mark tasks as completed or incomplete.
	•	Track pending and completed tasks. 

📝 Quiz System

	•	Create multiple-choice quizzes by subject. 
	•	Store questions, choices, and correct answers. 
	•	Take quizzes and receive automatic scoring. 
	•	Display quiz score and percentage after completion. 

📊 Progress Tracker

	•	View the total number of completed tasks. 
	•	View the number of pending tasks. 
	•	Monitor total study hours logged from quiz sessions. 
	•	Track overall academic progress.
